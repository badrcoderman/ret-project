# 🚀 مشروع RET: نظام الذكاء الاصطناعي المتكامل (AI System)

مرحباً بك في توثيق مشروع **RET**. يهدف هذا المشروع إلى تحويل بيئة التطوير والتحكم بالذكاء الاصطناعي من المحرر التقليدي (VS Code) إلى **لوحة تحكم ويب تفاعلية (Web Dashboard)** تعمل بالكامل داخل المتصفح بخصوصية تامة.

---

## 🌟 ما تم إنجازه (What We Did)

لقد قمنا ببناء نظام كامل يربط بين واجهة المستخدم (Frontend) ومحرك الذكاء الاصطناعي (Backend) مع الميزات التالية:

### 1. 🧠 محرك الذكاء الاصطناعي (AI Engine)
*   **الموقع:** `ai_engine/server.py`
*   **الوظيفة:** يعمل كعقل للنظام باستخدام Python و Flask.
*   **المميزات:**
    *   يدعم نموذج **GPT-2** للدردشة وتوليد النصوص.
    *   يحتوي على **جدار حماية (AI Firewall)** لتنقية المدخلات الضارة.
    *   يدعم **التدريب المحلي** على ملفات المشروع.

### 2. 📊 لوحة التحكم الذكية (Web Dashboard)
*   **الموقع:** `frontend/ai_dashboard.html`
*   **الوظيفة:** واجهة تحكم مركزية لإدارة المشروع.
*   **المميزات:**
    *   **💬 محادثة حية (Live Chat):** التحدث مباشرة مع الذكاء الاصطناعي.
    *   **⚡ التحكم بالتدريب:** زر لبدء تدريب النموذج مع شريط تقدم مباشر (Real-time Progress) باستخدام `Socket.IO`.
    *   **📥 استيعاب الكود (Code Ingestion):** ميزة لتحميل الكود من روابط خارجية (URL) وتدريب النظام عليها فوراً.
    *   **📂 مدير الملفات:** استعراض وتعديل ملفات المشروع من المتصفح.
    *   **🛠️ الإصلاح الذاتي (Self-Healing):** فحص ملفات المشروع واكتشاف الأخطاء تلقائياً.

### 3. 🎨 التصميم وتجربة المستخدم (UI/UX)
*   **التصميم:** تم اعتماد تصميم **Glassmorphism** (الزجاجي) العصري.
*   **التجاوب:** الموقع يعمل بكفاءة على جميع الأجهزة (Desktop, Tablet, Mobile).
*   **اللغة:** دعم كامل للغتين العربية والإنجليزية (RTL/LTR).

---

## ⚙️ كيف يعمل المشروع (How It Works)

يعمل النظام بتناغم بين مكونين رئيسيين:

1.  **الخادم الخلفي (Backend):**
    *   يتم تشغيله عبر `run_system.py`.
    *   يقوم بتحميل نموذج الذكاء الاصطناعي في الذاكرة.
    *   يفتح قنوات اتصال (WebSockets) لإرسال التحديثات الفورية للواجهة.

2.  **الواجهة الأمامية (Frontend):**
    *   تتصل بالخادم عبر `API` لإرسال الأوامر (مثل الدردشة أو طلب التدريب).
    *   تستقبل البيانات وتعرضها للمستخدم بشكل رسومي جميل.

---

## 🚀 كيفية التشغيل (How to Run)

لتشغيل النظام بالكامل، اتبع الخطوات التالية في التيرمينال:

1.  **تثبيت المكتبات اللازمة:**
    ```bash
    pip install flask flask-socketio transformers torch requests
    ```

2.  **تشغيل النظام:**
    ```bash
    python run_system.py
    ```

3.  **الدخول للوحة التحكم:**
    *   افتح المتصفح واذهب إلى: `http://localhost:5000/dashboard`
